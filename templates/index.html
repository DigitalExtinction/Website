Digital Extinction

Digital Extinction, or DE in short, is a multiplatform real-time strategy (RTS) game. It is free software and open source, licensed under GNU General Public License version 3 (GPLv3).

First version of the game is yet to be released. Currently, it is very incomplete and under continuous development. Please visit DigitalExtinction/Game GitHub repository to find more information.
